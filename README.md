# Employee Performance Analysis 

## Project Overview
This project focuses on predicting employee performance rating using various machine learning classification models. The goal is to compare multiple algorithms and identify the best-performing model. 
 ---

 ## Problme Statement
 Given employee-related features, predict the preformance Rating using supervised machine learning techniques

 ## Models Implemented
 1. Logistic Regression
 2. K-nearest Neighbours (KNN)
 3. Decision Tree
 4. Random Forest
 5. Support Vector Machine (SVM)
 6. Gradient Boosting

 ## Model Performance (Accuracy)

 Logistic : 86.25%
 KNN: 85%
 Decision Tree: 87.92%
 Random Forest: 94.58%
 SVM: 86.67%
 Gradient Boost: 93.33%

**Best Model: Random Forest (94.58%)
